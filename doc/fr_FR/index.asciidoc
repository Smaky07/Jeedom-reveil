= Gestions des reveil

== Description
Ce plugin permet de gérer facilement vos reveil.
Celui-ci est entirement basé sur le plugin Heliotrope qui est un prérequis pour l'utilisation de ce plugin.

Vous avez la possibilité de fermer vos reveil a la tomber de la nuit et de les ré-ouvrir au levé du soleil.
Vous avez egalement la possibilité de gérer vos reveil pour qu'il se ferme si le soleil est dans l'azimute de votre feunetre
